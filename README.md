## ✅ Tasks Completed

### 1. Contact Form — `index.html` + `style.css`
- Input fields: Full Name, Email, Phone, Message
- Styled using CSS with hover and focus effects
- Error states highlighted in red

### 2. JavaScript Form Validation — `validation.js`
- Name: checks for empty input
- Email: validated using regex pattern
- Phone: must be exactly 10 digits
- Message: checks for empty input
- Shows inline error messages and a success message on valid submit

### 3. Responsive Layout — `layout.html` + `layout.css`
- **Flexbox** used for the navigation bar
- **CSS Grid** used for sidebar + main content layout
- **Media queries** for mobile responsiveness (stacks vertically on screens ≤ 768px)

### 4. Dynamic To-Do List — `todo.html`
- Add tasks using input field or Enter key
- Mark tasks as done (strikethrough effect)
- Delete individual tasks
- Clear all tasks at once
- Live counter showing Total and Done tasks
- Built using pure JavaScript DOM manipulation
